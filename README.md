# Triglav's Vim files.

### Download

    $ git clone https://github.com/triglav/vim-files.git ~/.vim
    $ cd ~/.vim
    $ git submodule init
    $ git submodule update

### Installation

    $ ln -s ~/.vim/.vimrc ~/.vimrc

#### Windows tips

* use `~/vimfiles` instead of `~/.vim`
* create rather a hard link (without the `-s` flag) instead
